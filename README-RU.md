# Сайт мастерской — как этим пользоваться

## Что это
Статический сайт на Astro. Каталог товаров и услуги ремонта хранятся в
JSON-файлах — редактируешь файл, сайт пересобирается сам.

## Запуск локально
Нужен Node.js 18+ (лучше 20+).

```bash
npm install
npm run dev
```
Откроется на http://localhost:4321

## Где менять контент

- `src/data/products.json` — товары. У каждого: id, slug (для URL), name,
  category, brand, condition ("new"/"used"), price, inStock (true/false),
  warranty, image (путь к фото), description.
  Просто копируй существующий блок `{ ... }` и меняй поля под новый товар.

- `src/data/categories.json` — список категорий, которые видны в фильтре
  каталога и на главной.

- `src/data/services.json` — список услуг ремонта на странице /repair/.

- `src/data/site.json` — телефон, адрес, часы работы, ключевые слова для SEO.
  **Это первое, что нужно поменять на реальные данные перед деплоем.**

- Фото товаров кладёшь в `public/products/` и указываешь путь вида
  `/products/имя-файла.jpg` в поле `image` у товара.

## Как это станет "менять без кода"
Сейчас это JSON-файлы — их можно редактировать прямо на GitHub в браузере
(без установки чего-либо), не обязательно трогать код. Когда/если понадобится,
чтобы товарами управлял сам хозяин магазина без гита — можно перевести
products.json на Google Таблицу (я закладывал структуру данных с этим
расчётом, переезд будет несложным, скажи, когда актуализируется).

## Деплой (бесплатно)
Проще всего — Vercel или Netlify.

### Vercel
1. Залей проект на GitHub (создай репозиторий, запушь код).
2. Зайди на vercel.com, залогинься через GitHub, "Add New Project",
   выбери репозиторий.
3. Framework Preset — Astro определится автоматически.
4. Deploy. Получишь адрес вида your-project.vercel.app с HTTPS.

### Netlify — аналогично, "Add new site" → "Import from Git".

## После деплоя обязательно
1. Замени в `src/data/site.json` телефон, адрес, часы на настоящие.
2. Заведи и заполни **Google Business Profile** (Google Мой Бизнес) —
   для локального поиска типа "ремонт техники в Охтирке" это решает
   больше, чем сам сайт. Укажи тот же адрес/телефон, что на сайте.
3. Замени картинки-заглушки в `public/products/` на настоящие фото.
4. Если появится Google Sheet с товарами — скажи, подключим.
