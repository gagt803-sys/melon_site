# Как подключить редактирование через Google Таблицу

## Шаг 1 — создать таблицу
1. Зайди на **drive.google.com**
2. Загрузи файл `products-template.csv` (он рядом, в этом же архиве) — перетащи в Диск
3. Клик правой кнопкой по загруженному файлу → **Открыть с помощью → Google Таблицы**
   Файл откроется как обычная гугл-таблица — дальше `.csv` можно удалить с Диска, таблица уже отдельная.

## Шаг 2 — опубликовать таблицу (чтобы сайт мог её читать)
1. В таблице: **Файл → Опубликувати в Інтернеті** (Publish to web)
2. В выпадающем списке выбери не "весь документ", а конкретный лист (вкладку) с товарами
3. Формат — **Значення, розділені комами (.csv)**
4. Жми **Опублікувати**
5. Скопируй ссылку, которую покажет Google — она будет выглядеть примерно так:
   `https://docs.google.com/spreadsheets/d/ТУТ-ДЛИННЫЙ-ID/pub?output=csv`

Важно: "опубликовать" — не то же самое, что "доступ по ссылке". Именно пункт "Опублікувати в Інтернеті" даёт нужную прямую CSV-ссылку.

## Шаг 3 — колонки в таблице (уже готовы в шаблоне)
| Колонка | Что писать |
|---|---|
| Категорія | Точно как в списке категорий на сайте (например "Кабель", "Електроживлення") |
| Назва | Название товара |
| Ціна | Просто число, например `350` |
| Наявність | `в наявності` / `під замовлення` / `немає` |
| Стан | `нове` / `б/у` |
| Фото (ім'я файлу) | Имя файла из папки `public/products/`, например `hopestar-a60.jpg`. Пусто — будет заглушка |
| Опис | Описание товара |

Чтобы **добавить товар** — просто добавляешь новую строку снизу. Чтобы **убрать** — удаляешь строку. Порядок строк = порядок на сайте.

## Шаг 4 — подключить ссылку к сайту
На **vercel.com** → твой проект → **Settings → Environment Variables** → добавить:
- Name: `PRODUCTS_SHEET_URL`
- Value: ссылка из шага 2

Сохранить → сделать любой новый деплой (или просто git push), чтобы переменная подхватилась.

# Автообновление сайта (чтобы правки в таблице сами появлялись на сайте)

Без этого шага сайт обновится только когда ты сделаешь `git push`. С этим шагом — сайт сам проверяет таблицу каждые ~30-60 минут.

1. В Vercel: **Settings → Git → Deploy Hooks** → создать хук (Create Hook), название любое, ветка `main`. Vercel даст ссылку вида `https://api.vercel.com/v1/integrations/deploy/...`
2. Иди на **cron-job.org** (бесплатно, без карты) → регистрация → **Create cronjob**
3. URL — вставь ссылку хука из пункта 1
4. Периодичность — например "каждые 30 хвилин"
5. Сохранить

Всё, дальше и ты, и владелец магазина можете вообще не трогать код — правите таблицу, через полчаса-час само появится на сайте.

# Google Analytics (бесплатно)

1. Зайди на **analytics.google.com** → создать аккаунт → создать "властивість" (property) для сайта
2. Тип потока — **Веб**, вставь адрес сайта
3. Google даст **Ідентифікатор вимірювання** вида `G-XXXXXXXXXX`
4. Вставь его в `src/data/site.json`, поле `"gaId"` (сейчас там пусто — просто впиши между кавычками)
5. `git push` — аналитика заработает сразу

Смотреть статистику потом — на том же analytics.google.com, раздел "Звіти в реальному часі" (кто на сайте прямо сейчас) и "Залучення" (какие страницы смотрят).
